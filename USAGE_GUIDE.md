## üìã Main Features

### 1. Seller Registration
**Purpose**: Register sellers to build their trust profiles

**How to Use**:
1. Go to "Register Seller" tab
2. Fill in seller details:
   - Name (business or personal name)
   - Email address
   - Phone number (optional)
   - Platform (WhatsApp, Instagram, Facebook, Other)
3. Click "Register Seller"
4. **Important**: Save the generated Public Seller ID (format: SELLER-ABC-123)

**What You Get**:
- Unique Public Seller ID for easy sharing
- New seller profile (trust score starts as "NEW")
- Entry in the seller directory

### 2. Transaction Verification
**Purpose**: Compare seller promises with actual delivery/billing

**How to Use**:
1. Go to "Verify Transaction" tab
2. **Find the Seller**:
   - Search by name, email, or platform
   - Or enter their Public Seller ID
3. **Enter Seller Promises** (what they told you):
   - Product/service description
   - Price promised
   - Delivery time promised
   - Return policy promised
4. **Enter Actual Details** (from invoice/receipt):
   - Actual product received
   - Actual price charged
   - Actual delivery time
   - Actual return policy
5. Click "Verify Transaction"

**What You Get**:
- AI-powered mismatch detection
- Detailed analysis of discrepancies
- Trust score update for the seller
- Verification report with explanations

### 3. Trust Score Lookup
**Purpose**: Check a seller's trustworthiness before buying

**How to Find Sellers**:

#### Option 1: Search by Name/Email
1. Go to "Analytics" tab
2. Use the search box to find sellers by:
   - Business name (e.g., "Rajesh Electronics")
   - Personal name (e.g., "John Smith")
   - Email address
   - Platform type

#### Option 2: Public Seller ID
1. If you have a seller's Public ID (SELLER-ABC-123):
2. Enter it in the "Public Seller ID" field
3. Click "Search"

#### Option 3: Browse Directory
1. Go to "Directory" tab
2. Browse all registered sellers
3. Click on any seller to view their trust score

**What You Get**:
- Current trust score (0-100 or "NEW")
- Trust level (Excellent, Good, Fair, Poor)
- Total verifications performed
- Success rate percentage
- Verification history

### 4. Seller Directory
**Purpose**: Browse all registered sellers on the platform

**How to Use**:
1. Go to "Directory" tab
2. Browse the list of all sellers
3. See basic info: name, platform, trust score
4. Click on any seller for detailed trust metrics

### 5. Audit Logs (Logged-in Users Only)
**Purpose**: Track all your verification activities

**How to Use**:
1. Go to "Audit" tab (requires login)
2. View your complete activity history
3. Filter by:
   - Action type (verification, registration, etc.)
   - Date range
   - Severity level
4. Use pagination to browse through logs

### How Scores Are Calculated
- **60% Weight**: Recent verification performance (last 10 transactions)
- **40% Weight**: Historical success rate
- **Penalties**: Based on mismatch severity
  - High severity: -30 points
  - Medium severity: -15 points
  - Low severity: -8 points

## üö® Troubleshooting

### Common Issues

#### "Seller Not Found"
- Check spelling of seller name/email
- Try searching by different criteria
- Seller might not be registered yet

#### "No Trust Score Available"
- Seller is new (no verifications yet)
- This is normal for newly registered sellers
- Score will appear after first verification

#### "Verification Failed"
- Check internet connection
- Ensure all required fields are filled
- Try refreshing the page

#### "Login Issues"
- Clear browser cache and cookies
- Try incognito/private browsing mode
- Check if email is verified (for email/password accounts)

### Getting Help
- Check the system status in the sidebar
- Review audit logs for error details
- Contact support if issues persist

## üí° Tips for Best Results

2. **Use Exact Numbers**: Enter precise prices, dates, and quantities

3. **Encourage Verification**: Ask satisfied customers to verify transactions
4. **Monitor Your Score**: Regular check-ups help maintain quality

### For Safe Buying
1. **Check Multiple Sources**: Don't rely on trust score alone
2. **Look for Patterns**: Check verification history for consistency
3. **Start Small**: Try smaller transactions first with new sellers
4. **Verify Transactions**: Help build the community database
